Viewing Processes in JMX
Adding a JMX server

The Fuse JMX Navigator, shown in Figure 12, displays all of the processes running in your application and drives all interactions with the debugging features. Other areas of the Fuse Integration perspective adapt to display information related to the node selected in the explorer. The explorer's context menu also provides the commands needed to activate route tracing and to add JMS destinations.

Figure 12. Fuse JMX Navigator


By default, the Fuse JMX Navigator discovers all of the JMX servers running on the local machine and lists them under the Local Processes tree. You can add other JMX servers using a server's JMX URL.
