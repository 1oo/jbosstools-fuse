camel-sap

The camel-sap package consists of ten different SAP components.

The Remote Function Call (RFC) components support the sRFC, tRFC, and qRFC protocols. However, the IDoc components facilitate communication using the messages in IDoc format. Each component uses the SAP Java Connector (SAP JCo) library to facilitate bidirectional communication with SAP and the SAP IDoc library to facilitate the transmission of documents in the Intermediate Document (IDoc) format.

For more information, refer the section Configuring the camel-sap module
